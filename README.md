# React Todo List App

Simple React Todo List Application

## Running

1) Clone the project
2) Execute `npm install` at the root folder
3) Execut `npm start`

## What did I use?

* Typescript
* Eslit
* Prettier
* React Context API
* Functional Components
* React Hooks
